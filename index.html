CSV

Генератор из CSV

index_csv.html

Генерация QR-кодов из CSV-файлов или ручной ввод криптокодов. Подходит для создания новых этикеток на товары. Название товара берется из имени файла или вводится вручную.

Возможности

  • Загрузка нескольких CSV-файлов одновременно
  • Ручной ввод отдельных криптокодов
  • Массовый ввод списка кодов
  • Автоматическое удаление "_gtin..." из названия
  • Экспорт выбранных кодов в PDF для печати

Извлечение из PDF

index_pdf.html

Извлечение готовых DataMatrix кодов из PDF-файлов. Подходит для работы с уже сформированными кодами маркировки. Есть режим автопрокрутки для сканирования.

Возможности

  • Загрузка нескольких PDF-файлов
  • Извлечение изображений в высоком качестве
  • Автопрокрутка для последовательного сканирования
  • Горячие клавиши для быстрой работы
  • Экспорт выбранных кодов в PDF для печати

Формат этикеток для печати

Размер этикетки

60 x 40 мм (ширина x высота). Горизонтальная ориентация.

Количество на листе

3 столбца x 6 строк = 18 этикеток на одном листе А4.

Содержимое этикетки

QR/DataMatrix код сверху, название товара мелким шрифтом снизу.

Формат вывода

PDF-файл, готовый к печати на обычном принтере.

[QR]
