Изображения будут извлечены из PDF в исходном качестве

Горячие клавиши:

  • Ctrl + DOWN Прокрутка вниз (следующий код)
  • Ctrl + UP Прокрутка вверх (предыдущий код)
  • Ctrl + Space Автопрокрутка ВКЛ/ВЫКЛ
  • Ctrl + A Выбрать все для печати
  • Ctrl + D Снять выделение

Инструкция:

  1. Загрузите PDF файлы с DataMatrix кодами
  2. Кликните на код или чекбокс для выбора
  3. Нажмите "Экспорт в PDF" для создания документа А4
  4. Размер этикетки: 60мм × 40мм (горизонтально)
  5. На листе А4: 3 столбца × 6 строк = 18 этикеток
